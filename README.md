# سهم سكرينر — منصة الفرز الشرعي والفني للأسهم الأمريكية

منصة عربية بالكامل (RTL) لفرز الأسهم الأمريكية تجمع بين:

1. **الفحص الشرعي ونسبة التطهير** — فحص آلي تقديري على منهجية معيار AAOIFI رقم 21:
   - فحص النشاط الأساسي (استبعاد البنوك التقليدية، التأمين، الخمور، القمار، التبغ…)
   - الدين الربوي ÷ القيمة السوقية ≤ 30%
   - (النقد + الاستثمارات بفائدة) ÷ القيمة السوقية ≤ 30%
   - الدخل غير المباح ÷ الإيراد ≤ 5%
   - **نسبة التطهير** ومبلغ التطهير التقديري لكل سهم + حاسبة تطهير تفاعلية

2. **ثلاثة فلاتر جاهزة** (بنفس رموز Finviz):
   | الفلتر | الاستخدام | السلسلة الأصلية |
   |---|---|---|
   | صيد السيولة | مضاربة قصيرة | `sh_curvol_o500,sh_float_u50,sh_price_1to10,ta_changeopen_u10,ta_perf_10to-i10` |
   | الزخم / السوينق | متابعة الزخم الأسبوعي | `sh_curvol_o500,sh_float_o20,sh_price_1to10,sh_relvol_o1,ta_changeopen_u5,ta_perf_1w10o` |
   | الاستثمار طويل المدى | جودة + نمو + اتجاه | `cap_smallover,fa_curratio_o1,fa_debteq_u1,…,ta_sma200_pa` |

   إضافة إلى **منشئ فلاتر مخصص** (سعر، حجم، حجم نسبي، أسهم حرة، قيمة سوقية، تغير اليوم، تغير من الافتتاح، أداء الأسبوع).

3. **الأهداف والتوقعات** — محرك فني يحسب لكل سهم ولكل استراتيجية:
   - أهداف T1/T2/T3 بأساس فني معلن (ATR، نقاط الارتكاز، قمم 50 يوم/52 أسبوع، هدف المحللين)
   - وقف الخسارة ونسبة العائد/المخاطرة
   - الاتجاه العام ودرجة الفرصة (0-100) وتوقع نصي آلي

## التشغيل

```bash
npm install
npm run dev        # http://localhost:3000
npm run build && npm start   # إنتاج
```

## مصادر البيانات

- **Yahoo Finance** (افتراضي، مجاني): فرز مخصص + قوائم مالية + تاريخ سعري. لا يتطلب مفاتيح.
- **Finviz Elite** (اختياري): إن وضعت `FINVIZ_AUTH_TOKEN` في `.env` تُطبَّق فلاتر Finviz بصيغتها الأصلية حرفياً عبر تصدير CSV.
- **بيانات تجريبية**: تظهر تلقائياً (بشريط تنبيه واضح) إذا تعذّر الوصول للمصادر الحية، كي تبقى المنصة قابلة للتجربة.

## البنية

```
src/lib/yahoo/      عميل Yahoo (كوكي + crumb) + الفرز + القوائم + الشموع
src/lib/finviz.ts   مزوّد Finviz Elite الاختياري
src/lib/filters/    الفلاتر الثلاثة الجاهزة + محرك الشروط + بوابات الاستثمار
src/lib/shariah/    قواعد الفحص الشرعي AAOIFI + نسبة التطهير
src/lib/targets/    المؤشرات الفنية + محرك الأهداف والتوقعات
src/lib/demo/       بيانات تجريبية حتمية
src/app/api/        مسارا الفرز وتفاصيل السهم
src/app/            الواجهة (الرئيسية، الفرز، تفاصيل السهم)
```

## الحسابات والخادم الخاص (وضع SaaS)

عند ضبط `DATABASE_URL` (PostgreSQL) و`AUTH_SECRET` تتفعل تلقائياً:
- **حسابات المستخدمين** (تسجيل/دخول بجلسات آمنة) مع حقل خطة FREE/PRO جاهز للتسعير
- **قائمة المتابعة** مع مراقبة تغيّر الحكم الشرعي لكل سهم وتنبيه بارز عند تغيّره
- **سجل صفقاتي**: توثيق الصفقة من صفحة السهم بهدفها ووقفها، متابعة حية بشريط تقدم، وإغلاق موثق

بدون قاعدة بيانات تعمل المنصة بوضعها الكامل الحالي (بلا حسابات) — لا شيء ينكسر.

### النشر على خادم خاص بأمر واحد

راجع الدليل الكامل: [`deploy/README-SERVER.md`](deploy/README-SERVER.md)

```bash
cp deploy/.env.server.example .env   # ضع النطاق وكلمات السر
docker compose up -d --build          # تطبيق + PostgreSQL + TLS تلقائي + جدولة يومية
```

## إخلاء مسؤولية

المنصة أداة فرز آلية للأغراض التعليمية؛ **ليست توصية استثمارية**، والفحص الشرعي
**تقديري اجتهادي** مبني على القوائم المالية المتاحة ولا يغني عن الرجوع للهيئات
الشرعية المعتمدة. الأسواق عالية المخاطر — تحقّق دائماً قبل أي قرار.
